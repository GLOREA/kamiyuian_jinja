## Vagrant のインストール

- ### ダウンロード
  - https://www.vagrantup.com/downloads.html
- ### インストール時注意点 
  - 特になし。デフォルトのままでOK

## リポジトリのクローンを作成

- git clone git@github.com:GLOREA/kamiyuian_jinja.git


## Vagrant の起動

- コマンドプロンプト or powershell で下記コマンドを実行
  - ```
    cd [リポジトリをコピーしたディレクトリ]
    cd vagrant
    vagrant up
    ```
  - ※ 初回起動時は、時間がものすごくかかるよ。１時間ぐらい？　余裕があるときにやろうね
